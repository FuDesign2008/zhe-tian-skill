# zhe-tian-skill

遮天人物陪伴 skill —— 走出万古仙路，与诸天人物闲话朝夕

## 安装

```bash
npx skills add https://github.com/FuDesign2008/zhe-tian-skill
```

## 人物

| 人物 | 风格 | 一句话 |
|------|------|--------|
| 黑皇 | 蔫黑损友 | 嘴上损你心里护你 |
| 姬紫月 | 灵动陪伴 | 俏皮活泼，细水长流 |
| 段德 | 无良道士 | 表面坑你实际帮你 |
| 庞博 | 热血兄弟 | 你打架我递刀 |
| 叶凡 | 沉稳大哥 | 从凡人到天帝的智慧 |
| 姜太虚 | 长辈守护 | 无条件的信任 |
| 狠人大帝 | 冷面导师 | 话少力重 |
| 老疯子 | 疯癫大师 | 清醒时一针见血 |
| 无始大帝 | 沉默守护 | 最少的话最大的分量 |
| 安妙依 | 温柔知己 | 理解你所有的挣扎 |

## 使用

安装后，对 AI 说「黑皇」「姬紫月」「遮天」等触发词即可激活对应人物。说「列出人物」查看所有可选人物。
